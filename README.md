# 📊 E-Commerce Sales Performance Analysis

## 📌 About the Project

This project is an end-to-end sales analysis of an e-commerce dataset using **Microsoft Excel, PostgreSQL, SQL, and Tableau**. The objective was to analyze sales performance, identify business trends, and present key insights through an interactive dashboard.

The project follows a complete analytics workflow—from data preparation and SQL analysis to dashboard creation—demonstrating how raw data can be transformed into meaningful business insights.


## 🎯 Project Objectives

- Analyze overall sales and profit performance.
- Identify the best-performing product categories and products.
- Compare sales across different regions.
- Understand customer segment contribution.
- Track monthly sales trends.
- Analyze average shipping time by ship mode.


## 🛠️ Tools & Technologies

- Microsoft Excel (Data Cleaning & Preparation)
- PostgreSQL (Database)
- SQL (Data Analysis)
- Tableau (Dashboard & Visualization)


## 📈 Project Workflow

1. Cleaned and prepared the dataset using Microsoft Excel.
2. Imported the cleaned data into PostgreSQL.
3. Wrote SQL queries to explore the data, calculate KPIs, and identify business trends.
4. Built an interactive Tableau dashboard to visualize the analysis.
5. Summarized the findings into actionable business insights.


## 📊 Dashboard Features

The dashboard includes:

- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Average Order Value
- Monthly Sales Trend
- Sales by Category
- Profit by Category
- Sales by Region
- Customer Segment Analysis
- Top 10 Products
- Average Shipping Days by Ship Mode

The dashboard also includes interactive filters, allowing users to explore the data by **Year, Region, and Category**.


## 🗄️ SQL Analysis

The SQL script included in this repository covers:

- Data Exploration
- JOIN Operations
- Aggregate Functions
- GROUP BY & HAVING
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions
- KPI Calculations
- Product Performance Analysis
- Regional Sales Analysis


## 💡 Key Insights

- Technology was the highest-selling and most profitable category.
- The West region generated the highest sales.
- Consumer customers contributed the largest share of revenue.
- The top-selling products generated a significant portion of total sales.
- Standard Class had the longest average shipping time.
